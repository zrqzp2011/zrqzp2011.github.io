Termux Linux安装与Mobox配置指南

本指南详细介绍在Termux中安装Linux系统以及配置Mobox Windows模拟器的完整步骤，包含准备工作、安装方法、后续配置和常见问题解决方案。

在Termux中安装Linux系统

1

准备工作

  • 安装Termux（推荐从F-Droid下载）
  • 更新软件包：pkg update && pkg upgrade -y
  • 安装必要工具：pkg install curl wget git proot proot-distro -y
2

更换国内镜像源（可选）

为加快下载速度（尤其在国内），可替换Termux默认源为清华镜像源：

sed -i 's@^\(deb.*stable main\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main@' $PREFIX/etc/apt/sources.list

安装Linux系统

推荐

方法一：使用proot-distro

  1. 查看可安装的Linux发行版：proot-distro list
  2. 安装系统（以Ubuntu为例）：proot-distro install ubuntu
  3. 登录系统：proot-distro login ubuntu --user root -- bash -l

方法二：手动安装

  1. 下载rootfs（以Debian为例）：wget https://raw.githubusercontent.com/EXALAB/Anlinux-Resources/master/Scripts/Installer/Debian/debian.sh && bash debian.sh
  2. 启动脚本生成后运行：./start-debian.sh

后续配置（可选）

换源

进入Linux后替换为国内镜像（以Ubuntu为例）：

sed -i 's|http://archive.ubuntu.com|https://mirrors.tuna.tsinghua.edu.cn|g' /etc/apt/sources.list apt update

安装图形界面

安装XFCE4 + VNC：

apt install xfce4 tigervnc-standalone-server -y vncserver :1

使用localhost:5901连接VNC（默认密码需首次运行时设置）

汉化

安装中文语言包并设置环境变量：

apt install locales language-pack-zh-hans -y export LC_ALL="zh_CN.UTF-8"

进阶配置：GPU加速

若设备支持GPU（如Snapdragon），可尝试安装virglrenderer：

pkg install virglrenderer-android

注意事项

  • 无需ROOT：上述方法均基于proot，无需解锁设备权限
  • 性能限制：容器化环境性能低于原生Linux，建议至少配备Snapdragon 865+ CPU

在Termux中安装Mobox (Windows x86应用模拟器)

1

准备工作

  • 下载Termux和Termux-X11
  • 下载Input Bridge
  • Termux基础配置：
#授予存储权限 termux-setup-storage #更换清华镜像源（国内用户推荐） sed -i 's@^\(deb.*stable main\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main@' $PREFIX/etc/apt/sources.list pkg update && pkg upgrade -y
2

安装Mobox

方法一（官方推荐）：

#一键安装脚本 curl -s -o ~/x https://raw.githubusercontent.com/olegos2/mobox/main/install && ./x

➔ 选择安装选项时建议选第2项 (WoW64版本)

方法二（手动安装补丁版）：

#解压预编译包到指定目录 tar -zxf /sdcard/Download/mobox_patched_1.2.tar.gz -C /data/data/com.termux/files

启动与配置

启动Mobox

输入以下命令启动Mobox：

# 启动命令 mobox

图形界面设置

  • 分辨率调整：在Termux-X11设置中选择1280x720
  • 输入控制：通过Input Bridge加载键位配置文件(.ibp文件)
  • 性能优化（可选）：启用DXVK异步渲染提高帧率
#启用DXVK异步渲染 开始菜单＞Install＞dxvk_async＞dxvk-dev
